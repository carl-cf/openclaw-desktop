# 🦞 OpenClaw Desktop 快速入门卡片

**3 分钟上手指南** | v1.0.0

---

## 📥 1. 下载（1 分钟）

访问：https://github.com/carl-cf/openclaw-desktop/releases/tag/v1.0.0

选择您的系统：
- 🪟 **Windows**: 下载 `.exe` 文件
- 🍎 **macOS**: 下载 `.dmg` 文件
- 🐧 **Linux**: 下载 `.deb` 文件

---

## 🔧 2. 安装（1 分钟）

### Windows
双击 `.exe` → 下一步 → 安装 → 完成

### macOS
双击 `.dmg` → 拖到 Applications → 完成

### Linux
```bash
sudo dpkg -i openclaw_1.0.0_amd64.deb
```

---

## ⚙️ 3. 配置（1 分钟）

### 启动应用
双击桌面图标

### 环境检测
自动检测 → 等待完成 → 下一步

### 配置 AI 模型
1. 选择：**百度千帆**
2. 模型：**Qwen3.5-Plus**
3. API Key：[点击获取](https://cloud.baidu.com)
4. 下一步

### 配置通信渠道
1. 勾选：**启用飞书**
2. App ID：[从飞书开放平台获取](https://open.feishu.cn)
3. App Secret：[从飞书开放平台获取](https://open.feishu.cn)
4. 完成配置

---

## 🚀 4. 开始使用

### 启动服务
点击 **"▶️ 启动服务"**

### 测试对话
1. 打开飞书
2. 找到 AI 助手
3. 发送：`你好`
4. 等待回复
