## 📥 第一章：下载与安装

### 1.1 下载安装包

**方法 1：GitHub Releases（推荐）**

1. 访问：https://github.com/carl-cf/openclaw-desktop/releases
2. 找到 **v1.0.0** 版本
3. 根据您的系统下载：
   - **Windows**: `OpenClaw-Setup-1.0.0.exe`
   - **macOS**: `OpenClaw-1.0.0.dmg`
   - **Linux**: `openclaw_1.0.0_amd64.deb`

**方法 2：直接下载（开发版）**

```bash
git clone https://github.com/carl-cf/openclaw-desktop.git
cd openclaw-desktop
npm install
npm run dev
```

---

### 1.2 安装步骤

#### Windows 系统

1. **双击** `OpenClaw-Setup-1.0.0.exe`
2. 点击 **"下一步"**
3. 选择安装路径（默认 `C:\Program Files\OpenClaw Desktop`）
4. 点击 **"安装"**
5. 安装完成后勾选 **"启动 OpenClaw Desktop"**
6. 点击 **"完成"**

✅ **安装完成！**

#### macOS 系统

1. **双击** `OpenClaw-1.0.0.dmg`
2. 将 **🦞 OpenClaw Desktop** 拖到 **Applications** 文件夹
3. 打开 **Applications** 文件夹
4. **右键** → **打开**（首次运行需要）
5. 点击 **"打开"**

✅ **安装完成！**

#### Linux 系统

```bash
# Debian/Ubuntu
sudo dpkg -i openclaw_1.0.0_amd64.deb

# 或使用包管理器
sudo apt install ./openclaw_1.0.0_amd64.deb
```

✅ **安装完成！**

### 2.3 配置 AI 模型

#### 步骤 1：选择模型提供商

**推荐选择**：

| 提供商 | 推荐模型 | 价格 | 特点 |
|--------|---------|------|------|
| **百度千帆** | Qwen3.5-Plus | ¥0.02/1K | 中文优化，代码能力强 |
| **SiliconFlow** | MiniMax-M2.5 | $0.001/1K | 长上下文，多模态 |
| **OpenAI** | GPT-4o | $0.03/1K | 全能型，生态完善 |

**操作**：
1. 选择模型提供商
2. 选择具体模型
3. 点击 **"下一步"**

---

#### 步骤 2：输入 API Key

**获取 API Key**：

**百度千帆**：
1. 访问 https://cloud.baidu.com
2. 注册/登录账号
3. 控制台 → 应用列表 → 创建应用
4. 复制 API Key

**SiliconFlow**：
1. 访问 https://cloud.siliconflow.cn
2. 注册/登录账号
3. 控制台 → API Keys → 创建
4. 复制 API Key

**OpenAI**：
1. 访问 https://platform.openai.com
2. 登录账号
3. API Keys → 创建
4. 复制 Secret Key

**输入 API Key**：
1. 粘贴到输入框
2. 点击 **"验证 Key"**（可选）
3. 点击 **"下一步"**

**配置步骤**：

3. 点击 **"完成配置"**

**适用场景**：

## 🐛 第八章：故障排除

### 8.1 服务启动失败

**症状**：点击启动后显示错误

**解决方案**：
1. 查看日志区域的错误信息
2. 检查 API Key 是否正确
3. 检查网络连接
4. 重启应用

---

### 8.2 无法连接飞书

**症状**：飞书消息无法发送/接收

**解决方案**：
1. 检查飞书 App ID 和 Secret 是否正确
2. 检查飞书机器人权限配置
3. 重启飞书机器人
4. 查看飞书开放平台日志

---

### 8.3 API Key 无效

**症状**：提示"API Key 验证失败"

**解决方案**：
1. 重新获取 API Key
2. 检查 API Key 是否过期
3. 检查账户余额是否充足
4. 联系模型提供商客服

---

### 8.4 应用无法启动

**症状**：双击图标无反应

**解决方案**：

**Windows**：
1. 右键 → 以管理员身份运行
2. 检查杀毒软件是否拦截
3. 重新安装应用

**macOS**：
1. 右键 → 打开
2. 系统偏好设置 → 安全性 → 允许
3. 重新安装应用

**Linux**：
```bash
# 检查依赖
ldd /opt/OpenClaw\ Desktop/openclaw-desktop

# 重新安装
sudo apt reinstall openclaw-desktop
```

## 📝 附录

### A. 系统要求

| 系统 | 最低要求 | 推荐配置 |
|------|---------|---------|
| **Windows** | Windows 10 | Windows 11 |
| **macOS** | macOS 10.15+ | macOS 12+ |
| **Linux** | Ubuntu 20.04+ | Ubuntu 22.04+ |
| **内存** | 4GB | 8GB+ |
| **磁盘** | 500MB | 1GB+ |

### B. 端口占用

OpenClaw Desktop 使用以下端口：

- `18789` - 网关服务（必须）
- `3000-3002` - 开发服务器（仅开发模式）

**如果端口冲突**：
- 在设置中修改端口号
- 或关闭占用端口的其他程序

### C. 数据备份

**配置备份**：
```
Windows: %APPDATA%\openclaw-config
macOS: ~/Library/Application Support/openclaw-config
Linux: ~/.config/openclaw-config
```

**备份方法**：
1. 复制配置文件夹
2. 保存到安全位置
3. 需要时恢复
