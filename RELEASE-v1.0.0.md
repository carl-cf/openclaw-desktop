## 🎯 快速开始

### 第 1 步：下载安装包
访问 https://github.com/carl-cf/openclaw-desktop/releases/tag/v1.0.0

### 第 2 步：安装应用
双击安装包，按照向导完成安装

### 第 3 步：首次启动
双击桌面图标启动应用

### 第 4 步：环境检测
等待自动检测完成（如缺失组件会自动安装）

### 第 5 步：配置 AI 模型
1. 选择模型提供商（推荐：百度千帆）
2. 选择具体模型（推荐：Qwen3.5-Plus）
3. 输入 API Key

### 第 6 步：配置通信渠道
1. 勾选"启用飞书"
2. 输入飞书 App ID 和 Secret
3. 点击"完成配置"

### 第 7 步：启动服务
点击"▶️ 启动服务"按钮

### 第 8 步：开始对话
打开飞书，给 AI 助手发送消息
